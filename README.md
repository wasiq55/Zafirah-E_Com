<!-- ✨ Core Features -->
## ✨ Core Features

* **🔒 Secure Authentication:** Implemented robust JWT-based user authentication and authorization token systems to safeguard user data.
* **📦 Advanced Inventory Control:** Full product CRUD (Create, Read, Update, Delete) functionality seamlessly managing an active catalog of over 50+ products.
* **💳 Secure Payment Gateway:** Integrated Razorpay API for smooth checkout flows, real-time order processing, and reliable transaction handling.
* **🛒 Interactive Cart & Orders:** Real-time state management for handling product selections, quantities, and order tracking histories.
* **📱 Fully Responsive UI:** Optimized across all device types from mobile screens to desktop monitors for a frictionless shopping experience.
